Prime Framework

Prime Framework is a collection of projects that assist in building web applications (in Java and/or JavaScript). Each sub-project has its own website that contains a brief overview and usage for that project. Here's the list:

  • Prime JS

A core principle of Prime is that we love documentation INSIDE code. Therefore, our online documentation will be sparse. We encourage you to download the source code and use it as a starting point for Prime. However, some projects do require a bit more than reading documentation. For these projects, refer to the example on the website or any examples inside the repository itself.

Issues

Please log issues using the Github issue tracking system. Be sure to log issues to the correct project.

Mailing lists

Prime framework maintains a single email list for all projects. The email list is a Google group and you can join it here:

https://groups.google.com/forum/#!forum/prime-framework-users
